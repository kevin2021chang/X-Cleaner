*4**）**Access information about Wi-Fi networks*

This permission allows APP to view the user's current network status and execute corresponding advertising strategies.

*5**）**Clear the cache of all installed applications*

This permission is required to clear the cache of all applications installed in the garbage removal function on the user’s device.

*6**）**Broadcast after the system starts*

This permission is used to help App Lock to run automatically when the phone restarts.

*7**）**Expand or collapse the status bar*

This permission is only used to guide users who use the notification manager for the first time.

*8**）**Controllable vibrator*

This permission allows App Lock to provide vibration feedback after entering the wrong password.

*9**）**Terminate the background process*

We are allowed to stop other applications and use this permission to elevate your phone.

*10**）**Request the installation package*

This permission is only used in the App Manager function to help users easily install applications.

*11**）**Find out the space used by any package*

This permission is used to find the space occupied by any package, thereby providing garbage cleaning and App Manager functions.

*12**）**Prevent the processor from sleeping or the screen from dimming*

We can avoid the screen dimming during virus scanning and junk scanning.

*13**）**View the running application*

This permission is used to view the running applications to clean up memory usage.
